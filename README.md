## 1. Guide to run

In Terminal VSCode:

### 1.1 Frontend

<ul>
    <li>1. cd client</li>
    <li>2. yarn</li>
    <li>3. yarn start</li>
</ul>

### 1.2 Backend

<ul>
    <li>1. cd server</li>
    <li>2. add file .env includes: MONGO_URI (connect to your Database)</li>
    <li>3. yarn start</li>
</ul>
